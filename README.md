# ProyectoFInalIA2
Análisis visual usando YOLOv8. El sistema detecta jugadores y bola en un video aéreo de fútbol, compara modelos YOLOv8 y YOLOv8s, evalúa métricas de detección y genera un video MP4 final con bounding boxes y analíticas por frame, como conteo de jugadores por lado de cancha, jugadores fuera del campo, jugadores en áreas penales y posición del balón.
